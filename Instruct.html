Game Mechanics

Core Game Play:

Natural Disaster is a one to six player turn based strategy game.  The game is a competetion for points.  As a single player game the points can be used to compare against previous scores.  With multiple players the game is a battle for the most points.

All of the players are the presidents of local building companies. To earn points you must build structures, win contracts, and with multiple players compete for land.  To build structures you need to use resources.  You accumulate resources at the beginning of yourr turn.  Some structures are worth more points but they also cost more resources.  There is a tech tree that causes only one building type to be useable in the beginning of the game.  You can sell structures to other players for points and/or resources.

There is a limited amount of land for you to build on.  The game ends when all of the empty land has been built on. Contracts all have different requirements, like structures needed to fill the contract must all be adjacent to each other.  Once you have built a structure you can sell it to another player, trade it, demolish it, or use it to fulfill a contract.  You do not own properties if they have been used to fulfill contracts. You cannot sell or trade structures that you have used to fulfill a contract.  You can demolish a structure that you own at twice the point cost of the original structure.  Once you have demolished a structure you can build a new structure on the land.  If the player does not build a new structure the land will be useable by other players.

The game will end when all of the available land has been built on.  You will only be able to build on land that is adjacent to a road.  Roads and bridges cannot be destroyed.  You can build roads for a small amount of resources.  All roads must be connected. No points are accumulated from building roads or bridges. Some rivers can have bridges built over them. Bridges must be attached to roads.

The resource a player gets during his or her turn will be determined at random. Sometimes instead of getting a resource a natural disaster will occur. The natural disaster will cause one of several things to happen, like a loss of resources or the destruction of an existing building.

Game Flow:

The game will start by prompting for the number of players. Each player will then be asked to enter their name. After all the names have been entered the game will begin. Each player will take a turn in the order that the players entered their names.

The map size will be determined by the players. The map or playing field will be randomly generated. Most of the land will be buildable and one road will be in the map. The map may also have up to five rivers.

When a your turn starts you will be given either two units of a random resource or a natural disaster will occur. The resources are wood, steel, concrete, glass, and gold. Gold can be used to buy any of the other four resources and works essentially as a "wild-card" resource. A natural disaster can cause the loss or addition of resources, the destruction of structure(s), and limit a players turn.

After you receive the resource your score, and resource counts will be displayed. Your turn last's until you choose to end the turn. You can build, sell, trade, bid for a contract, and demolish buildings during your turn. You can only build a structure if you have the resources and meet the requirements.

You can sell a structure to any other player. Roads and bridges are not owned by players. The buyer will be asked whether they want to buy the offered property. If they decline, the seller can make an offer to another player. The buyer gets no points with the purchase. The seller will not lose the points he or she earned for building the structure. The seller can ask for any amount of points and or resources that the buyer has.

You can trade any structure or structures with any other player's structure(s). The tradee must agree to the trade. Trades only effect ownership, they do not effect points or resources. A structure used to fulfill a contract can not be traded.

There are always three available contracts. You will win any contract you bid for if you meet the contract's requirement. Once a bid is won the you will no longer own the property. When a bid is won the contract is replaced by a new contract. When you demolish a structure you lose twice the point value of the structure.

When you build the map will be checked to see if any land is still available to be built on. If there is no land left, the game will end. The player with the most points will be declared a winner. If there is a tie the winner will be determined based on the number of resources they have.

Random Map Generation

All maps will be generated randomly. All maps will have zero to five rivers and one road.

Multi-player:

One to six players can play Natural Disaster. All humen (non-computer/AI) players must use the same computer to play. In single player mode the you work towards a high score. For multiple players you will compete for the highest number of points. If the game ends with a point tie the winner will be determined by the number of resources left. With multiple players this game is strictly a free for all. With one player, changing the turn only allocates more resources. In single player mode the player cannot sell or trade structures.

GUI Objects

Setup Screen:

Player Number - This should define how many players there are. The only options should be 1, 2, 3, 4, 5, and 6.

Screen Size - This will allow the players to determine how many tiles will be in the game. There should be a minimum size and maximum size imposed.

Begin Button - This will cause the screen to be exited and the Player Name Entry Screen to be launched.

Player Name Entry Screen:

Player Name - A prompt for each player should be given where they can enter their name.

Continue Button - When pressed the name fields should be checked to insure they are all filled out. If they are not filled out the user should be given a warning and forced back to the screen. If all the fields are filled out this should launch the game / main game screen.

Main Game Screen:

Player Turn Button - The button should list the name of the current player. When clicked the players turn should end and the next player's turn and resource allocation will begin unless the game is over. If all the tiles are filled the game will end and the Credits/End Game Screen will Appear.

Demolish - When clicked you will be allowed to select a building to demolish.

Sell - When clicked you will be brought to the Sell Screen.

Trade - When clicked you will be brought to the trade screen.

Contracts - When clicked you will be brought to the contracts screen.

Sell Screen

Buyer - This field will be used to choose the buyer to make an offer to.

Points - This field will be used to ask for 0 - X # of points for the land.

Gold - This field will be used to ask for 0 - X # of gold units for the land.

Glass - This field will be used to ask for 0 - X # of glass units for the land.

Concrete - This field will be used to ask for 0 - X # of concrete units for the land.

Steel - This field will be used to ask for 0 - X # of steel units for the land.

Wood - This field will be used to ask for 0 - X # of wood units for the land.

Sell Button - When clicked the offer will be checked to see if the potential buyer has the right number of resources. If the buyer doesn't the seller will be given a list of what the player has and which fields were too high. If the buyer does have the resources the buyer will be prompted with the offer. If the buyer accepts, the resources and property will change ownership. The player will be returned to the main game screen.

Trade Screen

Trade Button - When clicked the tradee will be asked if they are willing to trade. If they agree the property will be traded. The player will be returned to the main game screen.

Contract Screen

Contract - Each of the three currently available contracts will be listed.

Bid button - When clicked the player will be prompted to choose what property to use for the bid. If the property meets the bid requirements the player will lose ownership of the property and gain the points.

Credits/End Game Screen

Score - The player's placement and scoring will be listed.

Credits - The game credits will be listed.
